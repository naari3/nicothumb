# nicothumb

Download nicovideo thumbnail.

## Usage

```
nicothumb v0.1.0
naari3
Download nicovideo thumbnail.

USAGE:
    nicothumb.exe <URL>

ARGS:
    <URL>    

OPTIONS:
    -h, --help       Print help information
    -V, --version    Print version information
```

### examples

```
$ nicothumb.exe https://www.nicovideo.jp/watch/sm9
$ nicothumb.exe https://nico.ms/sm9
$ nicothumb.exe sm9
```
